# automation-calculator

## Purpose

To help people make and communicate automation decisions quickly and effectively. 

## Setup
### Dependencies
* Ruby 2.0.0+
* Docker Engine 17.09.0+
* Docker Compose (Must support Compose 3.4 format)

### Install
`./go init`

## Use
* `./go shell`
* `./go start`
* `./go test`
* `./go help`
